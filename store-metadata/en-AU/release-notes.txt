--- Description ---
Plan your Australian migration journey with confidence using MigrateAU — the most comprehensive skilled migration app with real-time data from official government sources.

KEY FEATURES:
- SkillSelect Round Tracker - live invitation round results with per-occupation minimum points
- Points Calculator - calculate your SC 189/190/491 points instantly
- AI Migration Consultant (Aria) - get personalised visa advice powered by Gemini AI
- Occupation Cutoff Table - 144+ occupations with colour-coded competitiveness
- State Nomination Dashboard - SC 190 & 491 allocations across all 8 states/territories
- Processing Times - real-time visa processing time estimates
- Push Notifications - instant alerts for new rounds, policy changes, and state updates
- Salary Data - median earnings by occupation (sourced from Jobs & Skills Australia)
- In-App News Feed - curated migration news from official sources
- Round History - track trends across 8+ previous invitation rounds
- Dark Mode - clean, distraction-free interface

VISA TYPES COVERED:
1. SC 189 — Skilled Independent (points-tested)
2. SC 190 — State/Territory Nominated
3. SC 491 — Skilled Work Regional (Family Sponsored & State Nominated)
